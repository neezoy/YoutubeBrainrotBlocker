# Youtube Brainrot Blocker

A Firefox extension to take back control of your YouTube experience.

## Features

- **Block Shorts** — removes Shorts from all pages, hides the Shorts button in the sidebar, and redirects `/shorts/` URLs to the regular video player
- **Hide Recommended** — removes the "Up Next" sidebar on video pages
- **Redirect Home** — redirects youtube.com to your Subscriptions feed and hides the Home button from the sidebar

Each feature can be toggled independently from the extension popup.

## Installation

Install from the [Firefox Add-ons page](https://addons.mozilla.org/firefox/addon/youtube-brainrot-blocker) or load it manually:

1. Clone this repo
2. Go to `about:debugging` in Firefox
3. Click **This Firefox** → **Load Temporary Add-on**
4. Select `manifest.json`

## Requirements

Firefox 103 or later.
